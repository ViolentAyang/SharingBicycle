# Sharing Bicycle 项目

## 项目简介
Sharing Bicycle 是一个基于共享单车的安卓应用，包含用户注册登录、预约用车、查看行程、基于 GPS 的开锁、数据分析等功能。项目采用 C/S 架构，服务端通过 Java Servlet 实现，客户端为 Android 应用。

## 功能特点
- 用户注册与登录。
- 单车预约与开锁。
- 行程记录与数据分析。
- 钱包管理与积分系统。
- GPS 定位与导航。

## 技术栈
- **客户端**: Android, Java, XML。
- **服务端**: Java Servlet, Tomcat。
- **数据库**: MySQL。
- **其他**: JSON 数据传输。

## 使用方法
1. **服务端部署**:
   - 使用 Tomcat 部署 `sharing_bicycle_服务器` 目录下的项目。
   - 配置 MySQL 数据库，并导入初始数据。

2. **运行应用**:
   - 启动服务端后，打开客户端完成注册登录。
   - 体验共享单车功能。
